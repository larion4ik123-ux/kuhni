# CONTENT — материалы и контент

## Принципы

1. **Реальные фотографии клиента — основа сайта.** Не заменять стоковыми без
   необходимости. Не выдавать AI-визуализации за реальные работы.
2. **Контент редактируется в админпанели**, не зашивается в код.
3. **Без выдуманных данных.** Не придумывать стаж, число заказов, сотрудников.
4. **Отзывы только реальные** со ссылкой на источник.

## Структура assets/

| Папка | Содержание | Статус |
|---|---|---|
| `assets/raw/` | Оригиналы клиента | ⚠️ пока плейсхолдеры |
| `assets/processed/` | Варианты (thumbnail/card/hero/fullscreen, WebP+JPEG) | генерируется скриптом |
| `assets/owner/` | Фотография владельца (Артём Ермаков) | плейсхолдер |
| `assets/kitchens/` | Фото выполненных кухонь (работы) | плейсхолдеры |
| `assets/logo/` | Логотип | плейсхолдер |
| `assets/facades/` | Образцы фасадов (Interno, AGT, ...) | плейсхолдеры |
| `assets/colors/` | Образцы цветов фасадов | плейсхолдеры |
| `assets/handles/` | Образцы ручек/систем открывания | плейсхолдеры |
| `assets/countertops/` | Образцы столешниц | плейсхолдеры |
| `assets/reviews/` | Скриншоты отзывов (опционально) | пусто |

## Манифест

`assets/assets-manifest.json` — метаданные каждого файла: id, category,
ориентация, варианты, suggested_usage, quality, suitable_for_*,
caption/layout/style/primary_color/alt_text/focus_point/display_order/
is_real_work. Используется при seed-загрузке галереи и контента.

## Редактируемые тексты (site_blocks, ключи)

- `hero_title`, `hero_subtitle`, `hero_button_text`, `hero_note`,
  `hero_owner_name`, `hero_owner_role`, `hero_owner_quote`
- `advantages_*` (преимущества: индивидуальный проект, реальные материалы,
  изготовление под размеры, доставка и монтаж, работа по договору)
- `generator_*` (заголовок и шаги блока генератора)
- `about_*` (о компании: подход, материалы, контроль)
- `process_*` (этапы работы: 1-8)
- `materials_*` (бренды и группы материалов)
- `contacts_*` (телефон, Telegram, адрес, регион, часы, Яндекс Карты)
- `faq_*` (вопросы и ответы)
- `cta_*` (тексты кнопок и CTA)

## Подпись к AI-визуализации (обязательно)

«Визуализация является предварительной и служит для подбора идеи».
Не обещать точный проект или технический чертёж.

## Что можно подтвердить (из ТЗ, не выдумывая)

- Владелец: Артём Ермаков, лично контролирует проекты.
- Услуги: индивидуальный проект, реальные материалы, изготовление под размеры,
  доставка и монтаж, работа по договору.
- Варианты подбора: планировки (прямая/угловая/П-образная/с островом),
  стили (современный/неоклассика/классика), цвета (серый/бежевый/зелёный/белый),
  ручки (скоба/чёрная квадратная/кнопка/Gola), фасады (Interno/AGT/...),
  фурнитура (Tandembox/скрытые направляющие/Blum/Hettich/Boyard).

См. `docs/CONTENT_REQUIRED.md` — что нужно запросить у клиента.
